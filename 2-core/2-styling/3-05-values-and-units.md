## 3.5 Values and units

### 📝 나연

#### 길이

1. 절대 길이 단위
   cm / mm / Q / in / pc / pt / px

2. 상대 길이 단위
   em / ex / ch / rem / lh / rlh / vw / vh / vmin / vmax  
   vb / vi / svw / svh / lvw / lvh / dvw / dvh

- vh / vw ->viewport height and width

현재 실행중인 스크린 크기에 맞춰 상대적 크기를 반환!

- % 단위와의 차이점

```css
width: 100%;
width: 100vh;
```

vh / vw 는 열려있는 화면 전체의 상대 길이
% 는 % 가 선언된 요소의 부모 요소의 길이가 기준!
